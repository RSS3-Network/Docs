  An RSS3 Global Indexer (GI), is responsible for facilitating coordination
  among RSS3 Nodes, engaging with the VSL and performing critical functions such
  as Network Rewards calculation and slashing enforcement. Global Indexer API
  provides RSS3 Node data query (DSL) and network transparency (NTA) functions.
