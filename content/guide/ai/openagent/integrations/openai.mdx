## Introduction

OpenAgent is able to bootstrap your development with native OpenAI API support.

An API key from OpenAI is required, get your API key directly from OpenAI: https://platform.openai.com/.

After obtaining your API key, simply put it in your OpenAgent .env file, and you are good to go.
