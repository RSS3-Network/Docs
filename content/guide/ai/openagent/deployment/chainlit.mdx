## Introduction

[Chainlit](https://docs.chainlit.io/) is an open-source Python package to build production ready Conversational AI. For Chainlit’s configuration, refer to [Chainlit docs](https://docs.chainlit.io/).

OpenAgent uses Chainlit for demonstration purposes. You may use OpenAgent’s APIs to power your own applications.
