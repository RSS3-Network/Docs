## Introduction

Alpha Node is a type of [Normal Node](./normal-node).

Alpha Node is a special status awarded to RSS3 Nodes that joined the RSS3 Network during the Alpha Mainnet stage from March 12 to June 3, 2024.

The status indicates their early contribution and longstanding support for the Network.
